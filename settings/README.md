# settings

Multiplatform settings storage.

## Installation

Add dependency to your `build.gradle(.kts)` or `pom.xml`:

```kotlin
api("dev.kaccelero:settings:0.6.6")
```

```xml

<dependency>
    <groupId>dev.kaccelero</groupId>
    <artifactId>settings-jvm</artifactId>
    <version>0.6.6</version>
</dependency>
```
