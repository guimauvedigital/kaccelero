# auth

Reusable auth basics.

## Installation

Add dependency to your `build.gradle(.kts)` or `pom.xml`:

```kotlin
api("dev.kaccelero:auth:0.4.4")
```

```xml

<dependencies>
    <dependency>
        <groupId>dev.kaccelero</groupId>
        <artifactId>auth-jvm</artifactId>
        <version>0.4.4</version>
    </dependency>
</dependencies>
```
