# ktor-auth

Reusable auth basics.

## Installation

Add dependency to your `build.gradle(.kts)` or `pom.xml`:

```kotlin
api("dev.kaccelero:ktor-auth:0.1.1")
```

```xml

<dependencies>
    <dependency>
        <groupId>dev.kaccelero</groupId>
        <artifactId>ktor-auth-jvm</artifactId>
        <version>0.1.1</version>
    </dependency>
</dependencies>
```
